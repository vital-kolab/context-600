Context-600

Visual Behavior and Neural Responses under varying object context

What is Context-600?

Context-600 is a large primate dataset. It contains behavioral data from 300 human participants performing binary object discrimination while the image context with respect to the target object is varied. It also contains data from the same behavioral task performed by monkeys. Furthermore, it contains data from neural recordings performed on 2 rhesus macaques, each observing 600 images over multiple sessions. Our base images are selected across 10 object categories, and from the MS COCO dataset.

Diverse Images

We incorporate the two core values of neuroscience and computer vision into one dataset. From neuroscience, we have taken a careful setup of slow event-related to ensure a stimulus to reponse mapping. From computer vision, we have taken some of the most iconic and standard benchmark images. We have indoor/outdoor scenes and a lot of naturalistic images. They're complicated and contain object to object interaction. Explore our images and check out our rich data.

Neural and behavioral data from NHP

Context-600 is a novel, large-scale, neurobehavioral dataset. There are 600 images presented to each subject (NHP and humans).

Easy to Download

All data are readily available for download. We have human and NHP behavioral data, NHP physiological data, images used as stimuli, and corresponding annnotations. Download, explore, and share your results.
